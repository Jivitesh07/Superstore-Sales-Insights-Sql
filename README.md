# Superstore-Sales-Insights-Sql

# Project Overview
This project focuses on analyzing Superstore sales data using SQL to uncover business insights related to sales performance, customer segments, regional trends, and profitability.

# Tools & Technologies
- MySQL
- SQL (DDL & DML)
- CSV Dataset

# Dataset Description
The dataset contains order-level information including order dates, shipping details, customer data, product categories, sales, discounts, and profit.

# Key Objectives
- Design and create a structured relational database
- Import and clean CSV data using SQL
- Perform analytical queries to evaluate sales and profit trends
- Identify high-performing regions, products, and customer segments

# Key Analysis Performed
- Total sales and profit analysis
- Region-wise and category-wise performance
- Impact of discounts on profitability
- Customer segment analysis

# Outcomes
The analysis provides actionable insights to support data-driven decision-making and improve business performance.
